關於我

大家好！我是吳松濂，目前就讀於國立虎尾科技大學資訊管理系二年級。在資訊科技的學習旅程中，我熱衷於掌握程式設計、資料結構等技術，並期待將學習轉化為實際應用。

學歷背景

自高中時期開始，我便對電腦科學和資訊技術產生濃厚的興趣。大學階段選擇了資訊管理系，讓我進一步深入了解程式設計、資料分析等技術領域，並在各項專案中實踐這些技能。

經歷

在學期間，我積極參加學校的各種專案與活動，並自學多門線上課程以拓展技術面。曾經參與過校內的技術研討會，與來自不同領域的同學討論學習，獲得了不少見解。此外，也嘗試建立自己的個人作品集，展示平日學習的程式和專案，這也是這個網站的由來。

技能

  • 程式語言：掌握了 HTML、CSS、JavaScript，並有一定的 Python 基礎。
  • 專案管理：在校內專案中擔任小組成員，負責規劃及執行部分程式開發工作。
  • 工具與技術：熟悉 Git 版本控制工具，並有基本的資料庫知識。

興趣與啟發

我熱愛電競，特別是《英雄聯盟》，深受職業選手 Faker 的啟發。他不懈的努力讓我時刻提醒自己在技術學習上要保持韌性。這份熱情也促使我對電競背後的技術產生濃厚興趣，未來希望能將所學運用到電競產業中。
